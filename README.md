# 🧼 Clean To-Do (WIP)

A minimal to-do list built with **Vite.js** and **Material UI**.
You can add, edit, delete and toggle tasks between “To Do” and “Done”.

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Check/uncheck to move between lists
- Simple modal interface

---

## 🧩 Tech

- Vite.js
- React
- Material UI
- JavaScript (ES6+)

---

## ⚙️ Run locally

```bash
npm install
npm run dev
```
